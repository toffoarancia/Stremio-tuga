# 🇵🇹 Os Teus Filmes Tuga — Stremio Addon v4

Addon não oficial para o Stremio. Usa Puppeteer (Chromium headless) para extrair streams diretos sem anúncios.

## Instalar no Stremio

```
https://SEU-USERNAME-os-teus-filmes-tuga.hf.space/manifest.json
```

**Settings → Addons → Add addon** → cola o URL acima.

## Debug

- `/debug-catalog` — testa a extração do catálogo
- `/debug-stream?url=URL` — testa a extração de streams de uma página
